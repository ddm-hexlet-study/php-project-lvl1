# Console games project<br />
- Parity check<br />
- Calculator<br />
- Greatest common divisor<br />
- Arithmetic progression<br />
- Prime number<br /><br /><br />

## Requires 
- wp-cli/php-cli-tools: ^0.11.13<br />
- (dev) squizlabs/php_codesniffer: 3.*<br /><br /><br />

## Installation<br />
